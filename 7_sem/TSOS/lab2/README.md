# Лабораторная работа: Генерация и модуляция звуковых сигналов (WAV)

## Возможности
- Генерация сигналов:
  - синусоида, импульс (разная скважность), треугольная, пилообразная, шум (white/pink/brown)
- Полифония: суммирование нескольких моно сигналов, аккорды, гармонический ряд, стерео
- Модуляция: AM, FM, PM с модуляторами разных форм (sine/pulse/triangle/sawtooth)
- Сохранение в WAV (16-bit PCM; стерео и моно)

## Установка зависимостей
Установите пакеты из `requirements.txt` в выбранное окружение Python (3.9+):

```powershell
# из папки lab1
python -m pip install --upgrade pip ; pip install -r requirements.txt
```

Примечание: если `pip` не найден, используйте `py -m pip`.

## Запуск

```powershell
# из папки lab1
python main.py
```

Файлы будут сохранены в папку `out_wav`.

## Структура
- `signal_generators.py` — базовые генераторы
- `polyphonic_signals.py` — полифония
- `signal_modulation.py` — AM/FM/PM
- `wave_manager.py` — сохранение/загрузка WAV
- `main.py` — демонстрация и сохранение результатов

## Примечания
- Девиацию/глубину модуляции можно изменять в `main.py`.
- Если громкость искажается/клиппует, включите нормализацию (по умолчанию включена в сохранении WAV).
